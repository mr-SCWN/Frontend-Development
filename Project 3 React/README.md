# React

**SERVER:**

In the folder "nodetodo/server":

*Download:*
```
npm install
```

*Run server:*
```
node server.js
```

**CLIENT:**

In folder "nodetodo/client":

*Download:*
```
npm install 
npm install bootstrap

```
In the folder "nodetodo/client":

*Run client:*
```
npm start
```

*LocalHost:*
```
localhost:3000
# (but client starts running automatically)
```
