# Svetle

**SERVER:**

In folder "nodetodo/server":

*Download:*
```
npm install
```

*Run server:*
```
node server.js
```

**CLIENT:**

In folder "nodetodo/client":

*Download:*
```
npm install 
npm install --save-dev sirv-cli

```

*Run client:*
```
npm run dev 
```

*LocalHost:*
```
localhost:5000
```
